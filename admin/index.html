الدخول عبر GitHub — بدون قاعدة بيانات، بدون خادم

يُحفظ في متصفحك فقط لمدة هذه الجلسة (يُمسح عند إغلاق التبويب)، ولا يُرسل لأي مكان سوى GitHub مباشرة.

كيف أنشئ Token؟ (مرة واحدة فقط)
  1. افتح github.com/settings/tokens?type=beta
  2. اضغط "Generate new token" → اختر مستودع forsah فقط تحت Repository access
  3. فعّل صلاحية Contents: Read and write
  4. انسخ التوكن (يظهر مرة واحدة فقط) والصقه أعلاه
